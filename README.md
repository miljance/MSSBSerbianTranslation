# Serbian Translations for Subscription Billing

This app provides Serbian language translations for the Microsoft Subscription Billing Module in Business Central.

## Description

The Serbian Translations for Subscription Billing app is a translation extension that localizes the Microsoft Subscription Billing module for Serbian-speaking users. This extension translates all user interface elements, including:

- Page captions and fields
- Actions and menu items
- Messages and tooltips
- Reports and document labels

## Dependencies

This app requires the Microsoft Subscription Billing module (version 27.0.0.0 or higher) to be installed.

## Language

- Target Language: Serbian (Latin script) - sr-Latn-RS
- Source Language: English (US)
